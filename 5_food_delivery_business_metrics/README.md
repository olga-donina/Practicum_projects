# Анализ ключевых бизнес-метрик сервиса доставки еды «Всё.из.кафе»

## Задачи проекта
Подготовка SQL-запросов и BI-дашборда для анализа ключевых метрик сервиса доставки еды «Всё.из.кафе» в Саранске. Визуализированы DAU, конверсия, средний чек, LTV и Retention для оценки динамики клиентской базы и поведения пользователей.

## [Дашборд](https://datalens.yandex/8bjazn8hp1dmt)

## Описание данных
Данные продукта состоят из нескольких таблиц:
 - **analytics_events** — журнал аналитических событий/логи. 
 - **advertisement_budgets** — рекламные затраты по дням и каналам привлечения.
 - **partners** — справочник партнёрских сетей и их ресторанов.
 - **dishes** — справочник блюд.
 - **cities** — справочник городов.

## SQL Queries

Файл `sql_queries_delivery.sql` содержит SQL-запросы для расчёта ключевых метрик:
- DAU
- Conversion Rate
- Average Order Value
- LTV
- Retention Rate

## Вывод
1. DAU (активность пользователей)
 - В мае–июне наблюдались пики (04.05, 03–11.06) и резкие спады (15–16.05, конец июня).
 - В целом тренд на лёгкое снижение, особенно к концу июня.
 - Выраженный эффект будни/выходные: активность выше в рабочие дни.

2. Conversion Rate (CR)
 - Стабильный на уровне 29–30%, колебания в пределах 27–33%.
 - Пики (01.05, 03.06, 30.06) вероятно связаны с промо-активностями.
 - Провалы (16.05, 24–25.05, 18.06) могут быть связаны с багами или отсутствием акций.

3. Средний чек
 - В среднем: 130–159 руб.
 - Рост в июне (+8,7% к маю), но внутри месяца нестабильность.
 - Пик 31.05–07.06, затем снижение.

4. Retention Rate
 - Сильный отток после 1-го дня: теряется до 85% пользователей.
 - В июне удержание хуже, чем в мае.
 - После 3-го дня метрика стабилизируется на низком уровне (3–5%).

5. LTV ресторанов и блюд
 - Основные драйверы: «Гурманское Наслаждение» (170 тыс.руб.) и «Гастрономический Шторм» (165 тыс.руб.).
 - «Шоколадный Рай» сильно отстаёт (61 тыс.руб.).
 - Хиты-блюда: «брокколи в духовке» и «шашлыки в песто» (35–40 тыс. каждый).
 - Смешанная категория «рыба+мясо» наиболее востребована.
