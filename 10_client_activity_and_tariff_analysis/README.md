# Подготовка данных об использовании тарифных планов абонентами компании «Мегасеть»

## Задачи проекта
Заказчик - команда продукта компании «Мегасеть» — федерального оператора сотовой связи. 
Нужно получить информацию, как абоненты пользуются услугами компании с точки зрения двух тарифных планов:
- Выгрузить данные абонентов с информацией о ежемесячных объёмах услуг (суммарная длительность звонков, объём интернет-трафика, количество сообщений) и посчитать текущие траты клиентов — согласно тарифным планам.
- Для каждого тарифного плана рассчитать среднее значение трат клиентов на услуги связи. При этом нужно учитывать только тех абонентов, которые продолжают пользоваться услугами компании.
- Среди действующих активных абонентов найти тех, кто пользуется услугами сверх тарифа, и посчитать их средние расходы и среднее значение переплаты для каждого тарифа.

## План по выполнению проекта
1. Первичное знакомство с данными и проверка на ошибки.
2. Расчет статистики для каждого абонента.
3. Расчеты для заказчика.

## Описание данных
Для анализа используется база данных, которая содержит информацию об активности абонентов:
 - **users** cодержит информацию о пользователях;
 - **calls** содержит информацию о звонках;
 - **internet** содержит информацию о длительности интернет-сессий;
 - **messages** содержит информацию об отправленных сообщениях;
 - **tariffs** содержит информацию о тарифных планах.
