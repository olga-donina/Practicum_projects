# Анализ активности клиентов и тарифных планов компании «Мегасеть»

## Цели проекта
Проанализировать использование услуг связи клиентами компании «Мегасеть» (Smart и Ultra) на основе данных о звонках, интернет-трафике и сообщениях. Цель — выявить средние расходы клиентов, определить долю тех, кто использует услуги сверх тарифа, и оценить переплаты, чтобы помочь продуктовой команде обновить линейку тарифных планов и разработать специальные предложения для активных клиентов.

## План по выполнению проекта
1. Знакомство с данными и проверка на ошибки (пропущенные значения, дубликаты).
2. Подготовка данных об активности клиентов и их использовании услуг.
3. Решение аналитических задач с помощью SQL.
4. Формулирование выводов и подготовка данных для команды продукта.

## Описание данных
Для анализа используются данные, представленные в следующих таблицах:
 - **users** cодержит информацию о пользователях;
 - **calls** содержит информацию о звонках;
 - **internet** содержит информацию о длительности интернет-сессий;
 - **messages** содержит информацию об отправленных сообщениях;
 - **tariffs** содержит информацию о тарифных планах.
