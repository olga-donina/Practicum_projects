# Анализ данных для агентства недвижимости

## Цели проекта
Проанализировать рынок жилой недвижимости Санкт-Петербурга и Ленинградской области на основе данных сервиса «Яндекс Недвижимость», чтобы выявить наиболее перспективные сегменты недвижимости и сезонные тенденции для планирования бизнес-стратегии агентства.

## [Дашборд](https://datalens.yandex/vh5tjipys6wcg)

## План по выполнению проекта
1. Знакомство с данными в DBeaver.
2. Исследовательский анализ данных рынка недвижимости.
3. Решение ad hoc аналитических задач с помощью SQL.
4. Формулирование выводов и аналитических комментариев.
5. Доработка дашборда в DataLens.

## Описание данных
Для анализа используются данные, представленные в следующих таблицах:
 - **users** cодержит информацию об игроках;
 - **events** содержит информацию о покупках;
 - **skills** содержит информацию о легендарных умениях;
 - **race** содержит информацию о расах персонажей;
 - **country** содержит информацию о странах игроков;
 - **classes** содержит информацию о классах персонажей;
 - **items** содержит информацию об эпических предметах.
